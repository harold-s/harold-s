# intl-tel-input-french-translation

The aim is to create a translation for the library intl-tel-input (https://github.com/Bluefieldscom/intl-tel-input)
The English country list comes from intl-tel-input library.
The French country list comes from http://www.buddyweb.fr/extensions/ (ISO 3166-1 alpha-2 list)


There are two ruby scripts, both intelligently keeps the English name in brackets and deletes the French name in brackets :
- [intl-tel-input-french-translation.rb] aimed at creating the same javascript var allCountries than the library.
- [intl-tel-input-french-translation-js-replace.rb] generates the javascript code needed before loading intlTelInput() to have the menu in French
[intl-tel-input-french-translation.rb]:intl-tel-input-french-translation.rb
[intl-tel-input-french-translation-js-replace.rb]:intl-tel-input-french-translation-js-replace.rb
